# Shared

This folder is reserved for guidance, templates, diagrams, and standards that apply across multiple Alibaba Cloud blueprints.

Use this folder for content that should not belong to only one blueprint, such as:

- Shared architecture review standards
- Shared security review checklist
- Shared cost review checklist
- Shared tagging and naming standards
- Shared diagram style guidance
- Shared contribution guidance for blueprint authors

Keep blueprint-specific content inside `blueprints/<blueprint-name>/`.
