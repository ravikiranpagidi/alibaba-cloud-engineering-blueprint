# Professional-Level Architecture Preparation

## Skills Covered

- Enterprise landing zones
- Multi-region architecture
- Hybrid connectivity
- Migration planning
- Security and compliance
- Cost governance
- Data and AI platform architecture
- Architecture decision records

## Required Portfolio Artifacts

- One secure VPC diagram.
- One scalable web app design.
- One data lake or streaming design.
- One landing zone design.
- One DR plan with RTO/RPO.
- Three ADRs explaining tradeoffs.

## Common Senior Traps

- Designing features without ownership.
- Ignoring operational runbooks.
- Underestimating data transfer cost.
- Treating multi-region as automatic.
- Missing identity, logging, and rollback details.
