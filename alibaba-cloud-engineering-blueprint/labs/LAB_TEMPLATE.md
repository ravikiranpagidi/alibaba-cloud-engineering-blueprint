    ## Architecture

    List account, region, permissions, tools, and prior labs.

    This lab may create billable resources. Verify current pricing, free tier status, quotas, and regional availability in the Alibaba Cloud console before starting. Delete all resources in the cleanup section.

    ## Step-By-Step Instructions

    1. Create isolated resources.
2. Configure the service.
3. Deploy or test the workload.
4. Capture evidence.
5. Record assumptions.

    ## Validation Steps

    List the commands, console checks, URLs, metrics, or logs that prove the lab works.

    ## Cleanup Steps

    Delete every resource, including logs, snapshots, public IPs, databases, buckets, and temporary credentials.

    ## Troubleshooting

    - Permission denied: inspect RAM policy.
- Network timeout: check route tables, security groups, NACLs, and DNS.
- Unexpected cost: stop the lab and inspect billing details.
