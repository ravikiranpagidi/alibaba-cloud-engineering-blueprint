    ## Practice Questions

    - Explain the difference between a region and an availability zone.
    - Choose the right storage service for static assets, backups, and relational data.
    - Design least privilege access for a developer.
    - Troubleshoot an ECS instance that cannot connect to RDS.

    ## Scenario Questions

    - A company wants a low-cost website. Which services would you choose and why?
    - A private ECS instance needs internet updates. How would you design egress?
    - A database must be backed up and restored to a point in time. What should you verify?

    ## Common Exam Traps

    - Overlooking billing model differences.
    - Confusing security groups and route tables.
    - Missing regional availability.
    - Choosing administrator access instead of least privilege.
