# Monitoring And Observability

    ## Purpose

    Define signals, dashboards, alerts, logs, traces, and SRE practices for production workloads.

    ## What To Learn

    - CloudMonitor
- SLS
- Tracing Analysis
- Prometheus
- Grafana
- Alerting
- SLIs
- SLOs
- Error budgets

    ## Hands-On Practice

    - Create monitoring and alerting
- Write an SLO
- Create an incident runbook

    ## Interview Prompts

    - What is the difference between logging and monitoring?
- How would you reduce noisy alerts?

    ## Production Lens

    For every design, ask: What can fail? Who can access it? How is it monitored? How is it backed up? How much will it cost? How will it be cleaned up or decommissioned?
