# Storage

    ## Purpose

    Choose and secure storage services for web assets, backups, logs, data lakes, and enterprise retention.

    ## What To Learn

    - OSS
- Cloud Disk
- NAS
- Tablestore
- Hybrid Backup Recovery
- Archive storage
- Lifecycle policies

    ## Hands-On Practice

    - Host a static website on OSS
- Design a backup strategy
- Configure lifecycle and replication

    ## Interview Prompts

    - How do you secure object storage?
- When is object storage not the right database?

    ## Production Lens

    For every design, ask: What can fail? Who can access it? How is it monitored? How is it backed up? How much will it cost? How will it be cleaned up or decommissioned?
