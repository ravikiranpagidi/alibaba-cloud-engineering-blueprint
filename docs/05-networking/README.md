# Networking

    ## Purpose

    Design secure, segmented, observable, and scalable cloud networks.

    ## What To Learn

    - VPC
- vSwitch
- Route tables
- NAT Gateway
- EIP
- SLB
- NACLs
- VPN Gateway
- Express Connect
- PrivateLink
- Global Accelerator
- CDN

    ## Hands-On Practice

    - Build public/private subnets
- Design hub-and-spoke
- Plan hybrid connectivity

    ## Interview Prompts

    - How would you secure ingress and egress?
- How do you connect on-premises networks to Alibaba Cloud?

    ## Production Lens

    For every design, ask: What can fail? Who can access it? How is it monitored? How is it backed up? How much will it cost? How will it be cleaned up or decommissioned?
