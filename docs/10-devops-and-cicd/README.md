# DevOps And Automation

    ## Purpose

    Automate cloud infrastructure and application delivery with repeatable, reviewable workflows.

    ## What To Learn

    - Alibaba Cloud CLI
- ROS
- Terraform
- ACK deployment
- GitHub Actions
- IaC
- Environment promotion

    ## Hands-On Practice

    - Create Terraform VPC baseline
- Deploy to ACK
- Promote dev to test to prod

    ## Interview Prompts

    - Why use IaC?
- How do you separate dev, test, and prod safely?

    ## Production Lens

    For every design, ask: What can fail? Who can access it? How is it monitored? How is it backed up? How much will it cost? How will it be cleaned up or decommissioned?
