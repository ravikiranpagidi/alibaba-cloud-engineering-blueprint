# Core Services

    ## Purpose

    Understand Alibaba Cloud account structure, console navigation, resource organization, billing, and cross-cloud service mapping.

    ## What To Learn

    - What Alibaba Cloud is
- Common enterprise use cases
- Console basics
- Resource groups
- Tags
- Billing model
- AWS/Azure/GCP comparison

    ## Hands-On Practice

    - Create a resource taxonomy
- Design a tagging policy
- Map services across clouds

    ## Interview Prompts

    - How does RAM compare with AWS IAM?
- How would you organize resources for dev, test, and prod?

    ## Production Lens

    For every design, ask: What can fail? Who can access it? How is it monitored? How is it backed up? How much will it cost? How will it be cleaned up or decommissioned?
