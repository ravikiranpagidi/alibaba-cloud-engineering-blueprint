# Alibaba Cloud Learning Paths Roadmap

## Vision

Build a serious open-source Alibaba Cloud learning system that helps learners move from cloud fundamentals to enterprise-grade architecture, data platforms, AI-ready platforms, security governance, certification, and interviews.

## Roadmap Themes

| Theme | Outcome |
|---|---|
| Learning paths | Role-based journeys with roadmaps, labs, projects, and interview checks |
| Enterprise readiness | Security, governance, cost, monitoring, compliance, and operational review content |
| Hands-on depth | Labs and projects that map to real workloads |
| Architecture maturity | Reference patterns, tradeoffs, diagrams, ADRs, and cloud comparisons |
| Community quality | Issue templates, labels, contribution standards, launch plan, and future roadmap |

## Phase 1: Foundation Complete

- Root README, roadmap, contribution guide, code of conduct, security policy, changelog.
- Eight learning-path folders with roadmaps and README files.
- Templates for labs, projects, ADRs, security reviews, cost reviews, data pipelines, and AI platforms.
- Diagram catalog with Mermaid starter diagrams.

## Phase 2: Lab Expansion

- Add screenshots and CLI examples for each lab.
- Add Terraform versions for infrastructure labs.
- Add cleanup validation scripts.
- Add estimated cost ranges per region after verifying current pricing.

## Phase 3: Project Depth

- Convert sample projects into end-to-end implementation guides.
- Add architecture decisions and production readiness reviews.
- Add monitoring dashboards and runbooks.
- Add failure injection scenarios.

## Phase 4: Certification And Interview Pack

- Expand practice questions for ACA, ACP Cloud Computing, ACP Security, and ACP Big Data.
- Add mock interview tracks by role.
- Add senior-level system design prompts and scoring rubrics.

## Phase 5: Community And Automation

- Add Markdown linting and link checks.
- Add example Terraform modules and GitHub Actions workflows.
- Add contributor onboarding and monthly roadmap reviews.
