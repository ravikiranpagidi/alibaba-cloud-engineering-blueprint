# Cost Optimization

    ## Purpose

    Control spend without weakening reliability, security, or engineering productivity.

    ## What To Learn

    - Pay-as-you-go
- Subscription
- Reserved instances
- Savings plans
- Lifecycle policies
- Right-sizing
- Auto Scaling
- Tags
- Budget alerts

    ## Hands-On Practice

    - Use the cost checklist
- Tag a project
- Identify common cloud cost mistakes

    ## Interview Prompts

    - How would you reduce cost by 30 percent?
- What cost controls should be automated?

    ## Production Lens

    For every design, ask: What can fail? Who can access it? How is it monitored? How is it backed up? How much will it cost? How will it be cleaned up or decommissioned?
