# Databases

    ## Purpose

    Choose database services based on consistency, scaling, availability, workload shape, and migration requirements.

    ## What To Learn

    - ApsaraDB RDS
- PolarDB
- AnalyticDB
- Lindorm
- OceanBase
- DTS
- Backup and migration

    ## Hands-On Practice

    - Create RDS and connect from ECS
- Plan an Oracle migration
- Compare OLTP and OLAP services

    ## Interview Prompts

    - When would you choose PolarDB over RDS?
- How would you migrate a live database with minimal downtime?

    ## Production Lens

    For every design, ask: What can fail? Who can access it? How is it monitored? How is it backed up? How much will it cost? How will it be cleaned up or decommissioned?
