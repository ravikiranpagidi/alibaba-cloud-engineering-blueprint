# Interview Preparation

    ## Purpose

    Practice Alibaba Cloud explanations, troubleshooting, and senior architecture scenarios with structured answers.

    ## What To Learn

    - Fundamentals
- ECS
- OSS
- VPC
- SLB
- RDS
- ACK
- Function Compute
- MaxCompute
- DataWorks
- Security
- Cost
- Migration
- Troubleshooting

    ## Hands-On Practice

    - Answer beginner, intermediate, and advanced questions
- Practice scenario-based architecture answers
- Build a portfolio narrative

    ## Interview Prompts

    - How would you design a scalable web app?
- How would you explain Alibaba Cloud to a non-technical stakeholder?

    ## Production Lens

    For every design, ask: What can fail? Who can access it? How is it monitored? How is it backed up? How much will it cost? How will it be cleaned up or decommissioned?
