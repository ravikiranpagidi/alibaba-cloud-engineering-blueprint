# Data Engineering And Analytics

    ## Purpose

    Build batch, streaming, warehouse, lakehouse, and governance patterns on Alibaba Cloud.

    ## What To Learn

    - OSS data lake
- MaxCompute
- DataWorks
- Realtime Compute for Apache Flink
- AnalyticDB
- Hologres
- E-MapReduce
- Data ingestion
- Metadata governance

    ## Hands-On Practice

    - Build batch ETL
- Build real-time streaming
- Design medallion architecture

    ## Interview Prompts

    - How would you design a data lake on OSS?
- How would you handle late-arriving streaming data?

    ## Production Lens

    For every design, ask: What can fail? Who can access it? How is it monitored? How is it backed up? How much will it cost? How will it be cleaned up or decommissioned?
