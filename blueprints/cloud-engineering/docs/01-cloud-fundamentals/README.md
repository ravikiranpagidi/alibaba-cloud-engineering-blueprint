# Cloud Fundamentals

    ## Purpose

    Build the vocabulary required to understand cloud design before learning individual Alibaba Cloud services.

    ## What To Learn

    - What cloud computing is
- Public, private, hybrid, and multi-cloud
- Regions and availability zones
- Shared responsibility model
- Migration basics
- High availability and disaster recovery

    ## Hands-On Practice

    - Map a simple app to cloud components
- Identify failure domains
- Write a shared responsibility table

    ## Interview Prompts

    - What is the difference between HA and DR?
- What does the cloud provider manage versus the customer?

    ## Production Lens

    For every design, ask: What can fail? Who can access it? How is it monitored? How is it backed up? How much will it cost? How will it be cleaned up or decommissioned?
