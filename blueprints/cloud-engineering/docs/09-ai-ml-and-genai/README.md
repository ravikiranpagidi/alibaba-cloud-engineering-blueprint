# AI, ML, And GenAI

    ## Purpose

    Design AI-ready platforms that support analytics, ML training, model serving, vector search, RAG, monitoring, and governance.

    ## What To Learn

    - PAI
- Feature workflows
- Model deployment
- MLOps
- LLM application architecture
- Vector search
- RAG
- AI governance

    ## Hands-On Practice

    - Design an AI-ready platform
- Connect curated data to model workflows
- Document AI risks

    ## Interview Prompts

    - How do analytics, ML, and GenAI platforms share data safely?
- What controls should exist around model access?

    ## Production Lens

    For every design, ask: What can fail? Who can access it? How is it monitored? How is it backed up? How much will it cost? How will it be cleaned up or decommissioned?
