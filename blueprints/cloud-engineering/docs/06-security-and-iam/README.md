# Security And IAM

    ## Purpose

    Implement least privilege access, encryption, auditability, and compliance-ready security controls.

    ## What To Learn

    - RAM users, groups, roles, and policies
- MFA
- KMS
- Cloud Firewall
- Security Center
- ActionTrail
- Cloud Config
- Secrets management
- Encryption

    ## Hands-On Practice

    - Implement least privilege
- Write a security review
- Design audit logging

    ## Interview Prompts

    - How do RAM roles reduce key exposure?
- What logs would you enable for regulated workloads?

    ## Production Lens

    For every design, ask: What can fail? Who can access it? How is it monitored? How is it backed up? How much will it cost? How will it be cleaned up or decommissioned?
