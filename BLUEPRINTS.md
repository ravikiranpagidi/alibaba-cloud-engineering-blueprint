# Blueprint Catalog

This file catalogs the Alibaba Cloud blueprint collection in this repository. The main learning and engineering content is intentionally kept at the repository root so the GitHub main page shows the folders directly.

## Active Blueprint Folders

| Folder | Description |
|---|---|
| [00-alibaba-cloud-engineering-blueprint](00-alibaba-cloud-engineering-blueprint/) | End-to-end Alibaba Cloud engineering reference with docs, labs, projects, architecture patterns, templates, scripts, certification preparation, interview preparation, and wiki drafts. |
| [01-cloud-foundations](01-cloud-foundations/) | Beginner-friendly cloud foundations and Alibaba Cloud orientation. |
| [02-cloud-engineer-path](02-cloud-engineer-path/) | Infrastructure engineering with compute, storage, networking, databases, monitoring, labs, and projects. |
| [03-platform-engineering-path](03-platform-engineering-path/) | DevOps, platform engineering, CLI, Terraform, ROS, ACK, CI/CD, and automation. |
| [04-data-engineering-analytics-path](04-data-engineering-analytics-path/) | Data lake, batch processing, streaming, serving layers, governance, and analytics patterns. |
| [05-ai-ml-genai-path](05-ai-ml-genai-path/) | PAI, model serving, MLOps, RAG, vector search, GenAI architecture, and AI governance. |
| [06-security-governance-path](06-security-governance-path/) | IAM, RAM, security monitoring, audit, compliance, tagging, naming, and landing-zone controls. |
| [07-solution-architecture-path](07-solution-architecture-path/) | Reference architectures, decision guides, architecture reviews, case studies, and system design questions. |
| [08-certification-interview-path](08-certification-interview-path/) | Certification study paths, practice questions, scenario answers, flashcards, and mock interviews. |
