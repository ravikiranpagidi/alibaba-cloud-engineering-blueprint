# Alibaba Cloud Service Comparison

Service mappings are conceptual, not exact one-to-one equivalents. Capabilities, pricing, quotas, identity models, integrations, and regional availability may differ. Always verify with official cloud documentation.

| Capability | Alibaba Cloud | AWS | Azure | GCP |
|---|---|---|---|---|
| Virtual machines | ECS | EC2 | Azure VM | Compute Engine |
| Object storage | OSS | S3 | Blob Storage | Cloud Storage |
| Kubernetes | ACK | EKS | AKS | GKE |
| Serverless functions | Function Compute | Lambda | Azure Functions | Cloud Functions |
| IAM | RAM | IAM | Entra ID / RBAC | IAM |
| Load balancing | SLB | ELB | Azure Load Balancer / App Gateway | Cloud Load Balancing |
| Monitoring | CloudMonitor | CloudWatch | Azure Monitor | Cloud Monitoring |
| Data warehouse | MaxCompute / AnalyticDB | Redshift | Synapse | BigQuery |
| Data integration | DataWorks | Glue | Data Factory | Data Fusion |
| ML platform | PAI | SageMaker | Azure ML | Vertex AI |
| Logging | SLS | CloudWatch Logs | Log Analytics | Cloud Logging |
| DNS | Alibaba Cloud DNS | Route 53 | Azure DNS | Cloud DNS |
| Infrastructure as Code | ROS / Terraform Provider | CloudFormation / Terraform | ARM/Bicep / Terraform | Deployment Manager / Terraform |
| Container registry | Container Registry | ECR | Azure Container Registry | Artifact Registry |
| Audit trail | ActionTrail | CloudTrail | Activity Log | Cloud Audit Logs |
| Configuration governance | Cloud Config | AWS Config | Azure Policy | Config Controller / Asset Inventory |
| CDN | Alibaba Cloud CDN | CloudFront | Azure CDN / Front Door | Cloud CDN |

## Notes
