# Contributing

Thank you for helping improve Alibaba Cloud Learning Paths.

## Contribution Principles

- Keep content practical, enterprise-aware, and beginner-friendly.
- Add cost warnings, validation steps, and cleanup steps to every lab.
- Prefer diagrams, decision tables, checklists, and examples over long abstract prose.
- Verify current service names, pricing behavior, quotas, and regional availability from official Alibaba Cloud documentation.
- Avoid vendor marketing language; explain tradeoffs clearly.

## Content Checklist

- [ ] Purpose and audience are clear.
- [ ] Services used are named accurately.
- [ ] Security, cost, and monitoring are addressed.
- [ ] Cleanup is included for hands-on content.
- [ ] Interview explanation is included where useful.
- [ ] Links are relative when pointing inside the repository.
