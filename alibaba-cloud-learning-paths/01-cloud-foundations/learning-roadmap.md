# Cloud Foundations Learning Roadmap

## Outcome

By the end of this path, you should be able to explain, design, and validate Alibaba Cloud solutions related to Cloud computing vocabulary, IaaS PaaS SaaS, Regions and zones, Shared responsibility, Billing basics, Basic networking.

## Weekly Plan

| Week | Focus | Hands-On Output |
|---|---|---|
| 1 | Concepts and service map | Notes, diagrams, and flashcards |
| 2 | Core implementation patterns | At least two labs completed |
| 3 | Security, monitoring, and cost | Checklist and review notes |
| 4 | Project and interview practice | Portfolio project plus Q&A answers |

## Skills Checklist

- [ ] Explain Cloud computing vocabulary
- [ ] Explain IaaS PaaS SaaS
- [ ] Explain Regions and zones
