# Certification Guide

Alibaba Cloud certification names and exam objectives can change. Use this guide as a study structure and verify current requirements in Alibaba Cloud Academy.

## Study Method

1. Read service notes.
2. Complete the matching labs.
3. Explain each service out loud.
4. Practice scenario questions.
5. Review common traps.
6. Confirm current exam guide.

## Common Exam Traps

- Confusing security groups and network ACLs.
- Ignoring region and zone availability.
- Choosing public IPs where private networking is better.
- Forgetting cleanup and lifecycle cost.
- Treating object storage as a database.
- Overusing administrator permissions in RAM.
