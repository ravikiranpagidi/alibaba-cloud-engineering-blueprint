# Landing Zone Governance

## Minimum Enterprise Controls

- Account or resource-group structure for environments and teams.
- Tagging standard for owner, environment, application, data classification, and cost center.
- RAM least privilege and MFA.
- Centralized ActionTrail logs.
- Cloud Config compliance rules.
- Baseline networking with private subnets and controlled egress.
- KMS key ownership model.
- Budget alerts and cost allocation.
- Security review and architecture review gates.

## Compliance Checklist

- Identify data classification and residency.
- Define logging retention.
- Document encryption controls.
- Document access review cadence.
- Document backup and DR testing.
- Document incident response ownership.
