# Enterprise Architecture

    ## Purpose

    Translate business goals into secure, governed, resilient, and operable cloud architecture.

    ## What To Learn

    - Landing zones
- Multi-account strategy
- Shared services
- Network topology
- Centralized logging
- Security baseline
- Architecture review

    ## Hands-On Practice

    - Write an ADR
- Build an enterprise landing zone diagram
- Run an architecture review

    ## Interview Prompts

    - What belongs in a landing zone?
- How do you balance autonomy and control?

    ## Production Lens

    For every design, ask: What can fail? Who can access it? How is it monitored? How is it backed up? How much will it cost? How will it be cleaned up or decommissioned?
