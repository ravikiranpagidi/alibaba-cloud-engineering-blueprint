# Compute

    ## Purpose

    Learn when to choose virtual machines, containers, autoscaling, load balancing, and serverless compute.

    ## What To Learn

    - ECS
- Auto Scaling
- Server Load Balancer
- Function Compute
- ACK
- Image and instance lifecycle

    ## Hands-On Practice

    - Create ECS
- Deploy a load-balanced app
- Build a serverless API
- Deploy a container app

    ## Interview Prompts

    - When would you choose ECS over Function Compute?
- How do images, snapshots, and scaling groups work together?

    ## Production Lens

    For every design, ask: What can fail? Who can access it? How is it monitored? How is it backed up? How much will it cost? How will it be cleaned up or decommissioned?
