# Cloud Governance Checklist

## Identity

- [ ] Root account protected.
- [ ] MFA enabled for privileged access.
- [ ] RAM roles preferred over long-lived keys.

## Network

- [ ] CIDR plan documented.
- [ ] Private subnets for workloads.
- [ ] Controlled egress.

## Logging

- [ ] ActionTrail enabled.
- [ ] SLS retention defined.
- [ ] Security logs centralized.

## Cost

- [ ] Tags required.
- [ ] Budgets configured.
- [ ] Cleanup process documented.

## Compliance

- [ ] Data classification documented.
- [ ] Encryption requirements documented.
- [ ] Access review cadence documented.
